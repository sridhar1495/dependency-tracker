⚙ Connect to Live DependencyTrack API

API calls are routed through this dashboard's built-in nginx proxy — no cross-origin issues, no CORS configuration needed.

Enter the API key for your DependencyTrack instance. The key can be found in Administration → Access Management → Teams.

Need to generate an API key? Run:

TOKEN=$(curl -sf -X POST http://localhost:8081/api/v1/user/login \ -d 'username=admin&password=YOUR_PASS') \ && curl -sf -H "Authorization: Bearer $TOKEN" \ http://localhost:8081/api/v1/team | jq -r '.[0].apiKeys[0].key'
